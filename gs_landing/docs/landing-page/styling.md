# GardenSuite Landing Page - Styling Direction

## Visual Goal

The page should feel:

- light
- precise
- product-led
- calm
- established

It should not feel:

- dark
- glossy for its own sake
- overly scenic
- soft and repetitive
- assembled from many unrelated SaaS references

## Light Mode Only

This homepage should remain light-mode only.

That means:

- white is the base
- warm off-white is support
- green is accent
- dark blocks should be rare and purposeful, not the main system

## What To Keep

- some hero impact
- some tea-garden atmosphere if it does not bury the product
- clean minimal navigation
- restrained use of green

## What To Reduce

- repeated rounded cream cards
- decorative glows
- scenic layers that compete with product proof
- fake dashboard illustrations
- bento layouts used only because they are fashionable

## Layout Direction

- fewer sections
- more open white space
- larger full-width proof moments
- fewer nested containers
- clearer alternation between open editorial sections and product-detail sections

## Component Rules

### Cards

- use cards only when a card is genuinely needed
- reduce border radius from the current extra-soft feel
- avoid stacking page sections as giant cards unless the section truly needs framing

### Buttons

- primary button in green
- secondary button quiet and crisp
- no heavy glassmorphism
- no oversized pill treatments everywhere

### Navigation

- keep minimal
- support the hero instead of competing with it
- use transparency carefully

## Color Direction

Base palette direction:

- white base
- warm off-white secondary surfaces
- near-black text
- muted gray secondary text
- deep green accents for CTA and key highlights

Use color with restraint. The page should not feel washed in green.

## Product Proof Rules

- prefer real screenshots
- show real app states, not fake charts standing in for capability
- use real workflow visuals where possible
- every visual should help a buyer believe the product is real and already working

## Section Rhythm

Recommended rhythm:

1. high-impact hero
2. problem strip (tight, factual)
3. product grid (cards routing to product pages)
4. proof strip (stats + Sarbani Associates)
5. differentiator (3 trust cards)
6. FAQ
7. CTA

This gives each section a clear role without repeating the same layout.

## Motion

- subtle only
- support reading order
- no scenic motion that becomes the story
- product and copy should still work fully without motion

## Styling Test

If a section still makes sense after removing glow, blur, and decorative background effects, the structure is probably strong enough.
